# Fundamental Analysis Streamlit Application

## Installation Instructions
To install the required dependencies, you need to have Python installed. Then, install the required libraries by running:

```bash
pip install -r requirements.txt
```

## Usage Guide
Once the dependencies are installed, you can run the application with the following command:

```bash
streamlit run app.py
```

## Features
- Interactive charts for data visualization
- Real-time data fetching for stock analysis
- User-friendly interface to explore financial metrics

## How to Run on Replit
1. Go to [Replit](https://replit.com/) and create a new Python Repl.
2. Upload your project files including `app.py` and `requirements.txt`.
3. In the Replit shell, run the following command to install dependencies:
   
   ```bash
   pip install -r requirements.txt
   ```
4. Finally, start your Streamlit app by running:
   
   ```bash
   streamlit run app.py
   ```
5. Open the provided link to view your application in action!
